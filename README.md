## Installation

Copy the desired skill directories to your OpenCode skills folder:

```bash
# Windows
cp -r <skill-name> C:\Users\<username>\.config\opencode\skills\

# macOS/Linux
cp -r <skill-name> ~/.config/opencode/skills/
```

## Usage

Skills are automatically loaded by OpenCode. Refer to each skill's `SKILL.md` for specific usage instructions.

## Structure

Each skill follows this structure:

```
skill-name/
├── SKILL.md              # Main skill definition
└── resources/            # Optional supporting files
    └── *.md
```
