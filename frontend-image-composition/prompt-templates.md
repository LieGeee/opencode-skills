# GPT Image Prompt Templates

## Shared Requirements

Always include these constraints when generating UI-ready assets:

- transparent background
- no background
- isolated subject
- clean edges
- UI-ready asset
- high readability on web pages

## Decorative Asset

`[style direction], decorative UI element, transparent background, no background, isolated subject, clean edges, UI-ready asset, high readability on web pages, suitable for hero section or card header, not a full scene`

## Character Or Mascot Asset

`[style direction], half-body or compact mascot illustration, transparent background, no background, isolated subject, clean edges, UI-ready asset, high readability on web pages, expressive but not noisy, suitable for frontend page support`

## Icon Or Functional Element

`[style direction], single UI icon or functional visual element, transparent background, no background, isolated subject, clean edges, UI-ready asset, high readability on web pages, strong silhouette, clear semantics`

## Replacement Asset

`[style direction], replacement for an existing frontend asset that currently feels AI-heavy or visually inconsistent, transparent background, no background, isolated subject, clean edges, UI-ready asset, high readability on web pages, controlled detail, production-friendly`

## Prompt Writing Notes

1. Keep the subject count low unless the page clearly needs a composed set.
2. Favor silhouette clarity over decorative density.
3. Avoid vague “futuristic glowing artwork” wording unless the product already uses that visual language.
4. Mention the exact placement when useful, such as hero, empty state, card header, side panel, or onboarding helper.
